# Professional Portfolio - Mayal Chauhan

A premium, fully dynamic portfolio website built with React, TypeScript, and Tailwind CSS. This project features a sophisticated admin dashboard for managing all content, including projects, blog posts, testimonials, and global site settings.

## üöÄ Features

- **Dynamic Content Management**: Full CRUD operations for projects, skills, experience, and biography.
- **Advanced Blog System**: A feature-rich blog with a visual editor, multimedia support (images/videos), and real-time interaction tracking.
- **Admin Analytics**: A comprehensive dashboard displaying visitor statistics, user growth, and interaction metrics using visual charts.
- **Global Settings Control**: Manage social media links, contact information, and resume downloads through a unified interface.
- **Premium UI/UX**: Responsive design with liquid animations, glassmorphism, and a festival-themed decor system.
- **Visitor Tracking**: Intelligent system to track guest and user visits with daily breakdowns.

## ‚öôÔ∏è Local Development

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory and add your credentials:
   ```env
   VITE_FIREBASE_API_KEY=your_api_key
   VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
   VITE_FIREBASE_PROJECT_ID=your_project_id
   VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
   VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
   VITE_FIREBASE_APP_ID=your_app_id
   VITE_CLOUDINARY_CLOUD_NAME=your_cloud_name
   VITE_CLOUDINARY_UPLOAD_PRESET=your_upload_preset
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```
